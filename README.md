# TP Git - Metodología de Sistemas I

Repositorio grupal que documenta los comandos de Git vistos durante la cursada.

## Contenido

- Explicaciones de comandos Git en archivos Markdown individuales
- [indice.md](/indice.md) con introducción y enlaces a todos los archivos
- [estadisticas.md](/estadisticas.md) con métricas del repositorio

## Metodología

El trabajo fue desarrollado siguiendo buenas prácticas de trabajo colaborativo:
Trabajo por ramas independientes y una rama "dev" para la integracion antes de subirla a "main", pull requests con revisiones, y mensajes de commit bajo la convención de Conventional Commits.
