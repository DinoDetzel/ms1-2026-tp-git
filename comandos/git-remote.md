## Conexión con la Nube

```
    git remote add origin <url_repo_remoto>
```
Importante: de ser la primera vez que vamos a hacer push al repositorio	remoto de los archivos de nuestro repositorio local, necesitamos primero vincularlos. Esto lo hacemos con "git remote add origin <url_repo_remoto>", siendo <url_repo_remoto> el link del repositorio remoto. Por convencion, el repositorio remoto y el local se les pone el mismo nombre. Es necesario hacer primero este comando para que git sepa a dónde queremos hacer el push.
