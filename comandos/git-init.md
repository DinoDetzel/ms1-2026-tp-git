# Creación del repositorio local

```
    git init
```
Crea una carpeta oculta llamada ".git/" dentro de la carpeta donde nos encontremos que almacena toda la configuracion y funcionamiento de git (historial, configuracion, ramas, commits, staging area). Hacer git init dentro de determinada carpeta, hace que esa carpeta sea tu repositorio local. Y a partir de ese momento git comienza a trackear todos los cambios realizados
