# Instalacion de Git.

## Instalación Windows:
```
	winget install --id Git.Git (opcional)
```
Usa winget, herramienta de la linea de comandos de microsoft para instalar, actualizar y desinstalar paquetes. Alternativa mas rapida para instalar git desde la terminal

---

## Instalación Linux(Ubuntu):
```
	sudo apt install git
```
*Equivalente de winget install pero para Ubuntu, sirve para instalar git en Linux desde la terminal.

---

## Comandos de git.
```
	git --version
```
Sirve para verificar la instalacion de git, muestra la version de git que instalamos
