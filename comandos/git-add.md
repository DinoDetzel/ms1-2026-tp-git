## Paso previo a guardar cambios

```
    git add
```
Si queremos mover ciertos archivos del working directory al staging area, antes de formalizar el guardado, usamos git add. 

```
    git add "archivo"
```
Podemos especificar qué archivo especifico queremos pasar al staging area como por ejemplo "git add archivo.md".

```
    git add .
```
Si queremos seleccionar todos los archivos modificados con "git add ."
Aclaración: dentro de la carpeta oculta ".git/" se registran e identifican los archivos a los cuales se les realizaron cambios.
