## Información del repositorio

```
    git status
```
Nos comunica acerca del estado del repositorio. En qué rama estamos, cuantos commits llevamos, si hay algun archivo del cual no llevamos registro y los archivos que estan listos para hacerles commit. Siempre nos sugiere comandos add o rm por si queremos llevar un archivo desde el working directory al staging area y visceversa.
