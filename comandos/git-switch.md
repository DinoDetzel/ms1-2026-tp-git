## Cambiar de rama

```
    git switch <nombre_rama>
```
Alternativa moderna y mas segura que checkout. De la misma manera que con checkout "git switch <nombre_rama>" nos ubicara en la rama elegida. 

```
    git switch -c <nombre_rama>
```
Tambien permite crear una rama y moverse a ella con "switch -c <nombre_rama>".

---

Aclaración: funciona solo en las versiones mas nuevas de git.
